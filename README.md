# AI Market Assistant
ChatGPT-style AI chatbot that provides live crypto and stock data with interactive charts.
## Features
- ChatGPT-style interface
- Live stock & crypto prices
- Auto asset detection
- Interactive charts
- Real-time updates
- Professional fintech UI

```bash
git clone YOUR_LINK
